Baixar Ícones

Como usar estes ícones:

  1. Clique nos botões acima para baixar os arquivos de ícone
  2. Salve os arquivos como icon-192x192.png e icon-512x512.png
  3. Coloque os dois arquivos de ícone na mesma pasta do seu projeto
  4. Certifique-se de que os nomes dos arquivos correspondam exatamente aos especificados no manifest.json
  5. Após adicionar os ícones, seu aplicativo estará pronto para ser instalado como PWA!
